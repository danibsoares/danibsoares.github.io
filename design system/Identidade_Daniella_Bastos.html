02 · Essência Filosofia de design
Posicionamento

Editorial. Tipografia como protagonista. Cor como pontuação.

01

Tipografia gigante

Pesos altos (700–800) e tamanhos generosos. A palavra é a imagem.

02

Assimetria intencional

Quebra de grid, blocos coloridos, sobreposições. Nada centralizado por padrão.

03

Cor pontual

Lime neon, rosa, lavanda. Aparecem em flashes — nunca em massa contínua.

04

Pills e arcos

Cantos arredondados nas tags e botões. Tudo o mais é reto e firme.

— Princípios de uso
// VOZ DE DESIGN
"Editorial. Tipografia como
 protagonista. Cor como pontuação."

// REGRAS
1. Cream = 60% das telas.
2. Ink (texto) = 25% das telas.
3. Acentos somados = no máx 15%.
4. Saffron só no ponto da marca
   e em micro-detalhes.

// TIPOGRAFIA
- Display: pesos 700-800,
  letter-spacing -2% a -4%
- Serif italic: APENAS em palavras
  pontuais dentro de display.
  Nunca em corpo.
- Body: 14-16px, weight 400.
- Mono: 11-13px, uppercase,
  letter-spacing +6%.

// COMPONENTES
- Bordas: pill (9999px) ou
  card (16px). Nada com 4-8px.
- Botões SEMPRE pill.
- Cards SEMPRE 16px ou 24px.
- Border-width: 1.5px (não 1, não 2).

// LAYOUT
- Assimetria > simetria.
- Quebra de grid intencional.
- Negative space generoso.
- Numeração visível (01/04).
